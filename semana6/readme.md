# GALERÍA DE IMÁGENES INTERACTIVA - SEMANA 5

## DESCRIPCIÓN GENERAL
Aplicación web interactiva desarrollada con JavaScript puro que permite gestionar imágenes desde cualquier URL pública de internet. Implementa manipulación dinámica del DOM, manejo de eventos y validación de datos.

## FUNCIONALIDADES PRINCIPALES

### AGREGAR IMÁGENES
- Ingreso de URLs de imágenes desde cualquier fuente pública
- Validación automática de formatos (JPG, PNG, GIF, WebP, BMP, SVG)
- Soporte para servicios como Unsplash, Pexels, Pixabay
- Feedback visual inmediato

### SELECCIONAR IMÁGENES
- Sistema de selección única (una imagen a la vez)
- Resaltado visual con efectos CSS
- Indicador de selección visible
- Deselección con click o botón específico

### ELIMINAR IMÁGENES
- Eliminación de imagen seleccionada
- Animaciones de eliminación
- Actualización automática del estado
- Opción para limpiar galería completa

### CARACTERÍSTICAS ADICIONALES
- Imágenes de ejemplo precargadas
- Atajos de teclado (Enter, Delete, Escape)
- Diseño responsive para todos los dispositivos
- Animaciones CSS en transiciones
- Validación en tiempo real
- Contadores de estado

## INSTRUCCIONES DE USO

### AGREGAR UNA IMAGEN
1. Ingresa una URL válida de imagen en el campo de texto
2. Presiona el botón "Agregar" o la tecla Enter
3. La imagen se cargará y aparecerá en la galería

### SELECCIONAR UNA IMAGEN
1. Haz click en cualquier imagen de la galería
2. La imagen se resaltará con un borde azul
3. Un indicador de check aparecerá en la esquina

### ELIMINAR UNA IMAGEN
1. Selecciona una imagen haciendo click en ella
2. Presiona el botón "Eliminar Seleccionada"
3. O presiona la tecla Delete del teclado

### ATAJOS DE TECLADO
- ENTER: Agregar imagen desde el input
- DELETE: Eliminar imagen seleccionada
- ESCAPE: Deseleccionar imagen actual
- CTRL + A: Enfocar campo de URL

### BOTONES ESPECIALES
- EJEMPLOS: Carga imágenes de muestra automáticamente
- LIMPIAR: Elimina todas las imágenes de la galería
- DESELECCIONAR: Quita la selección actual
